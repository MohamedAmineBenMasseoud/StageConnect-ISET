# StageConnect ISET

Premium internship management platform prototype built with:

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- Chart.js
- Font Awesome
- Google Fonts (Poppins)

## Included modules

- Landing page with animated counters, testimonials, FAQ, and CTA sections
- Student dashboard, profile simulation, favorites, CV filename storage
- Company dashboard, internship publishing form, and applicants table
- Internship search with live filters and details page
- Reviews with star rating and localStorage persistence
- Notification center with unread badges and mark-as-read actions
- Simulated chat with typing effect and auto replies
- Admin dashboards with analytics tables and Chart.js graphs
- Print-ready internship report examples for all departments

## Local behavior

This project is fully static. It does not require Node.js or a backend.

All interactive demo data is stored in the browser using `localStorage`, including:

- favorites
- notifications
- profile data
- reviews
- custom internships
- applications
- chat messages
- CV filename simulation

## Editable areas

- Internship seed data: `assets/js/script.js`
- Department and level config: `assets/js/script.js`
- Report examples: `pdf/example_report.html`
- Company text blocks: `company/dashboard.html`, `company/add_stage.html`

Open `index.html` in a browser to use the prototype.
